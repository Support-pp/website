Unsere Features

Ticket System

Das System von Support++ bietet ein umfangreiches Ticketsystem. Dabei kann ein eigener Command festgelegt werden. Die Supporter werden bei einer Einsendung eines Tickets benachrichtigt und auch der Versender erhält eine Bestätigung für seine Versendung. Das Ticketsystem kann auf weitere Plattformen wie Discord und Telegram erweitert werden.

Supporter Notification

Das System von Support++ ermöglicht es auch, einer gewählten Supportergruppe einen oder mehrere Supporterchannel zuzuweisen. Dabei gibt es auch Funktionen, Supporter oder kleine Spielverderber auf eine Ignorelist zu setzen, damit Supporter nicht beim betreten von denjenigen benachrichtigt werden. Das besondere an unserem System ist, dass es auch möglich ist, die Benachrichtigungen auf weitere Plattformen wie Discord oder Telegram zu übertragen! Wir bieten auch eine Themen greifendes System, mit dem voreingestellte Fragen bereits vordefiniert werden können, damit sich ein Supporter bereits auf den Support einstellen kann.

Automatic Opening/Closing

Das System von Support++ bietet auch eine automatische Öffnungszeiten Funktion. Dabei kann der Benutzer selber die Öffnungszeiten verwalten und auch einstellen, ob und wie die Supportchannel bearbeitet werden sollen. Der Benutzer kann auch Commands zur Schließung der Supportchannel einstellen und ebenfalls festlegen, ob die Supporterchannel geöffnet sind, ob ein Supporter online, offline oder AFK ist. Ist das nicht toll? Wie in einem Supermarkt!

Servergroup Support Ban

Grade bei größeren Servern kann es hilfreich sein Nutzergruppen vom Support auszuschließen oder auch nur von bestimmten Support bereichen. Genau deshalb haben wir eine Möglichkeit eingebaut für jeden Warteraum eigenständig Gruppen festzulegen welche Ignoriert werden. Natürlich werden die Support Gruppen die für den Channel selber zuständig sind von Haus aus Ignoriert.

Wartemusik

Um die Wartezeit für die Wartenden so angenehm wie möglich zu gestalten stellen wir euch Wartemusik mit Ansagen zur Verfügung. Aber ihr könnt natürlich auch eigene Hochladen und Verwenden. Dabei entscheidet Support++ sogar zwischen Support Online und Offline.

Themen Bezogener Support

Da wir aus Erfahrung wissen das nicht jeder Supporter auch zu allen Themen ausgebildet ist, haben wir einen Themen Modul fest eingebaut wo mit man Themen festlegen kann, wo durch dann der dafür zuständige Supporter benachrichtigt wird.

Feedback System

Um vorallem bei großen Servern wo man schnell die Support Qualität aus den Augen verlieren kann, ist eine Rückmeldung von Nutzern wichtig. Somit kann man Supporter zu mehr disziplin anhalten oder die Schwarzen Schafe des Supports finden kann. Beim Feedback System können die Nutzer 1-5 für ein Supporter vergeben.

Discord Ticket Antwort PRO

Da grade für Leute die dass Supporten nicht Hauptberuflich machen auch die Zeit eine Rolle spielt, wissen wir das man nicht dauerhaft auf dem TeamSpeak Server sitzen kann. Hier für haben wir auch die Antwort auf Tickets über Discord ermöglicht, da aber das ganze über eine API die bei uns liegt läuft, können wir das nicht ganz ohne eure finanzielle Hilfe zur Verfügung stellen
