Warum wir?

Ticket System

Das System von Support++ bietet ein umfangreiches Ticketsystem. Dabei kann ein eigener Command festgelegt werden. Die Supporter werden bei einer Einsendung eines Tickets benachrichtigt und auch der Versender erhält eine Bestätigung für seine Versendung. Das Ticketsystem kann auf weitere Plattformen wie Discord und Telegram erweitert werden.

Supporter Notification

Das System von Support++ ermöglicht es auch, einer gewählten Supportergruppe einen oder mehrere Supporterchannel zuzuweisen. Dabei gibt es auch Funktionen, Supporter oder kleine Spielverderber auf eine Ignorelist zu setzen, damit Supporter nicht beim betreten von denjenigen benachrichtigt werden. Das besondere an unserem System ist, dass es auch möglich ist, die Benachrichtigungen auf weitere Plattformen wie Discord oder Telegram zu übertragen! Wir bieten auch eine Themen greifendes System, mit dem voreingestellte Fragen bereits vordefiniert werden können, damit sich ein Supporter bereits auf den Support einstellen kann.

Automatic Opening/Closing

Das System von Support++ bietet auch eine automatische Öffnungszeiten Funktion. Dabei kann der Benutzer selber die Öffnungszeiten verwalten und auch einstellen, ob und wie die Supportchannel bearbeitet werden sollen. Der Benutzer kann auch Commands zur Schließung der Supportchannel einstellen und ebenfalls festlegen, ob die Supporterchannel geöffnet sind, ob ein Supporter online, offline oder AFK ist.
